How to submit your map?

We will only ask you for a URL where we can see your map. It's up to you whether you want to use GitHub pages from your project repo, bl.ocks.org or even set up your own map service!

If for instance you want to use bl.ocks.org, you can create directly your final map in your Gist Account following these instructions, or even use something like blockbuilder.org, which simplifies the process a bit.
