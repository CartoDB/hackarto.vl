How to submit your map?

For better visualization, we have decided to use bl.ocks.org application. It is a simple viewer for sharing code examples hosted on GigHub Gist.

For experienced users, you can create directly your final map in your Gist Account following these instructions. This is an example.

For not experienced users, we have found a very easy tool for creating your map, it is called blockbuilder.org.

Go there, click "Start coding" and the builder will appear. Log in with your GitHub account in order to save your map when it is ready. Here it is an example how a map could look like.

Advices:

  • –  We encourage you to make your dataset(s) public, in order to not use an API key.
  • –  Unfortunately it is necessary an API key to make CARTO.js library work, but it is not necessary to use a real one if your dataset is public, so you can use a fake one.
